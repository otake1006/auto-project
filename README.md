## 🔍 開発で工夫した点 / Key Points

- ### 🎯 責務分離と保守性
  - ゲームロジック、描画、エフェクト演出を**独立したクラス構成**で管理
  - 例：`Character.ts`・`EffectManager.ts`・`BattleSystem.ts` など

- ### 🎲 拡張可能なゲーム設計
  - スキル・バフ・レリックなどを**データ駆動で実装**（JSON or データベース）
  - 「〇〇の条件で□□が発動」のような複雑な挙動を**汎用的に記述可能**

- ### 🌐 マルチプレイ対応
  - Colyseusを使ってリアルタイム同期
  - 再接続・観戦・セッション管理にも対応
